**API Test NASA**

Дополнительный учебный проект, чтобы расширить практику работы с API.  
Тесты выполняются на примере **NASA Open API**, чтобы познакомиться с другим видом сервисов и их спецификой.

**Цель проекта**

- Освоить основы автоматизации тестирования API с другим сервисом.
- Научиться работать с **Mocha** и **Chai** для написания тестов.
- Использовать **Axios** для отправки HTTP‑запросов.
- Попробовать формирование отчётов с помощью **Mochawesome**.

**Структура проекта**

- `test/` — директория с тестами.
- `config.js`, `consts.js`- вспомогательные файлы для конфигурации и констант.  
- `service.js` — модуль для работы с NASA API.  
- `mochawesome-report/` — автоматически генерируемые отчёты после запуска тестов.

**Установка**

Клонировать репозиторий и установить зависимости:

```bash
git clone https://github.com/09Natalia/api-test-nasa.git
cd api-test-nasa
npm install
```

**Запуск тестов**
```bash
npm test
```
